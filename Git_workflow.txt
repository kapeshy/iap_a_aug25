Git Workflow for iap_a_aug25 Project

1. Initialize a local repository (if not already initialized):
   git init

2. Check the status of your repository:
   git status

3. Stage changes for commit:
   git add .         # adds all modified files
 

4. Commit your staged changes with a message:
   git commit -m "Implemented validation of email address"

5. Set up the remote repository (only once):
   git remote add origin <https://github.com/kapeshy/iap_a_aug25.git>

6. Push your local commits to the remote repository:
   git push -u origin main 

7. Create a new branch for new features or bug fixes:
   git checkout -b <branch-name>

8. Merge a branch back to main after testing:
   git checkout main
   git merge <branch-name>
